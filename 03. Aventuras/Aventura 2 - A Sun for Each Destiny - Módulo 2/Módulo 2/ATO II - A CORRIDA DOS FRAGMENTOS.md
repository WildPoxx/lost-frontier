# ATO II — A CORRIDA DOS FRAGMENTOS

## Módulo 2 — Título a Definir

---

## Função do Ato para o Mestre

O Ato II é o ato da escolha sob pressão.

Depois do Mainframe, os PCs têm informação suficiente para entender que o satélite não é apenas salvage orbital. Mas a informação vem com urgência: o Energy Core pode estar sendo removido, o Sinal Eco pode ter sido reivindicado pelos Tarekhs, e Shizuka pode ter encontrado uma força capaz de transformar seu luto em operação militar.

Este ato deve ser modular. Os PCs podem tentar seguir para o Core, buscar o Sinal Eco, perseguir Shizuka, reforçar reconstrução de Outpost-87 ou dividir esforços. O mestre deve preservar os Focal Points, não a ordem rígida das cenas.

---

# KEY SCENE 1 — O MAPA NÃO FECHA

## Elos de Entrada

- dados extraídos do Mainframe;
- falha parcial no DT-Mainframe;
- ou sucesso forte que mostra lacunas deliberadas.

## Função da Cena para o Mestre

Transformar a recompensa do Ato I em nova escolha. O Mainframe fornece verdade suficiente para abrir o mapa, mas não para encerrar o mistério.

## Abertura Narrativa

Os dados parecem coerentes por alguns segundos. Depois começam as lacunas: transmissão confirmada, vetor incompleto, referência a Outpost-87 quebrada, sinal de energia branco extrapolando limites de segurança, e um canal de eco repetindo pacotes sem conteúdo limpo.

## Informações Reveláveis

| Camada | Informação |
|---|---|
| Mínima | o Core é mais perigoso que valioso |
| Provável | o Sinal Eco pode confirmar transmissão e nome do satélite |
| Profunda | duas peças bastam para direção aproximada de Outpost-87 |
| Opcional | o Mainframe registra uma tentativa de transmissão posterior à queda |

## Decisão dos PCs

Os jogadores devem escolher:

- ir ao Core;
- buscar o Sinal Eco;
- perseguir rastros de Shizuka/Hell Raiders;
- concluir reconstrução com risco;
- dividir grupo;
- pedir ajuda externa.

## Consequências de Ritmo

| Escolha | Efeito provável |
|---|---|
| Core primeiro | chance de interceptar extração, mas Sinal Eco avança |
| Sinal Eco primeiro | melhora Outpost-87, mas Core avança |
| Perseguir Shizuka | pode revelar Hell Raiders, mas atrasa fragmentos |
| Reconstruir dados ali | possível avanço de Outpost-87, mas clocks rivais avançam |

# KEY SCENE 2 — O SINAL BRANCO SAI DO CHÃO

## Local

Ponto do Energy Core ou trilha até ele.

## Função da Cena para o Mestre

Mostrar que o Core é objeto de corrida física. A cena pode ocorrer no local da queda, no caminho, ou por evidência indireta.

## Quatro Estados Possíveis do Core

| Estado | Condição | Cena resultante |
|---|---|---|
| Intocado | relógio 0-2 | PCs podem investigar e preparar extração |
| Contestado | relógio 3 | Hell Raiders ou Shizuka chegam junto |
| Em remoção | relógio 4-5 | Chase, sabotagem ou ataque ao comboio |
| Ausente | relógio 6 | investigação de rastro e virada para ruínas Anunnaki |

## O que o Core revela

| Camada | Informação |
|---|---|
| Mínima | há Aethera concentrada em escala perigosa |
| Provável | remover o Core sem contenção altera o terreno |
| Profunda | o Core responde a padrões Anunnaki, não apenas humanos |
| Opcional | o simbionte de Jax reage como se reconhecesse alimento ou parente |

## Desafios Mecânicos

- Hazard de Aethera: Fatigue, Wounds ou interferência.
- Dramatic Task de contenção se PCs tentarem manipular o Core.
- Chase se os Hell Raiders estiverem removendo.
- Combate se a negociação falha ou se Séquito força escalada.

## Elos de Saída

- Core protegido temporariamente;
- Core perdido para os Hell Raiders;
- Core rastreável até ruínas;
- evidência de Shizuka;
- primeiro contato direto com um Raider não infectado.

# KEY SCENE 3 — A MARCA DOS TAREKHS

## Local

Ponto do Sinal Eco / Antena-Telemetria, ou caminho até ele.

## Função da Cena para o Mestre

Resolver o terceiro fragmento como problema social e político, não como combate automático.

Os Tarekhs devem ser agentes com código, território e interesse. Alguns podem ver a peça como presságio, troféu de queda, prova de domínio, dívida de caça, item de troca ou ameaça a ser enterrada.

## Situação Inicial

O fragmento não está simplesmente abandonado. Há marcas, vigias, rastros de arrasto, bandeiras, ossos de metal empilhados ou outro sinal de posse.

## O Fragmento

O Sinal Eco é consolidado como:

- antena de transmissão;
- módulo de telemetria;
- carcaça externa com marca legível do Starwalker-IX;
- possível Cubo-Eco ou núcleo repetidor de sinal.

## Social Conflict — Reivindicar Leitura do Sinal Eco

### Objetivo dos PCs

Obter acesso suficiente para escanear, recuperar ou negociar a peça sem transformar os Tarekhs em inimigos permanentes.

### Skills Possíveis

| Skill | Uso |
|---|---|
| Persuasion | acordo, pacto, respeito |
| Intimidation | demonstração de força sem massacre |
| Common Knowledge | entender código local |
| Survival | provar leitura de território |
| Athletics/Fighting | duelo ritual ou prova física |
| Science/Electronics | demonstrar perigo técnico da peça |
| Performance | se houver rito, canto, prova pública ou storytelling |

## Outcomes

| Resultado | Consequência |
|---|---|
| Vitória limpa | acesso ao fragmento e possível aliança/aviso Tarekh |
| Vitória com custo | acesso limitado, dívida, tributo, promessa ou duelo futuro |
| Empate tenso | PCs podem escanear, mas não levar |
| Falha | Tarekhs barram acesso; outra rota técnica será necessária |
| Falha crítica | fragmento é movido, danificado ou entregue a terceiro |

## Revelações do Sinal Eco

| Camada | Informação |
|---|---|
| Mínima | marca visível: Starwalker-IX |
| Provável | transmissão partiu para fora do Campo |
| Profunda | vetor aponta para leste / Véu |
| Opcional | `Outpost-87` aparece menos corrompido que no Mainframe |

# KEY SCENE 4 — SHIZUKA ENTRE OS SAQUEADORES DE TUMBAS

## Natureza da Cena

Cena flexível. Pode ser direta ou indireta.

## Formas possíveis

- PCs veem Shizuka com Hell Raiders.
- PCs encontram mensagem dela para os Raiders.
- Um Raider menciona "a Kanei".
- O corpo de Haikyo aparece como carga ritual, barganha ou trauma.
- A katana de Jax reage quando Shizuka está perto.

## Função da Cena para o Mestre

Mostrar que Shizuka deixou de ser apenas fugitiva. Ela se reconstrói em outro ecossistema de violência.

## Estado Emocional de Shizuka

Ela não deve estar simplesmente "mais malvada". Ela está:

- enlutada;
- humilhada;
- endurecida;
- tentando dar sentido à morte de Haikyo;
- procurando força externa;
- disposta a aceitar uma leitura falsa se essa leitura prometer retorno ou vingança.

## Revelações

| Camada | Informação |
|---|---|
| Mínima | Shizuka encontrou uma força armada |
| Provável | ela trocou dados do SWIX por ajuda |
| Profunda | Haikyo se tornou parte do interesse ritual/technorgânico |
| Opcional | Séquito alimenta a crença de que Haikyo pode voltar |

# KEY SCENE 5 — O PRIMEIRO RAIDER QUE HESITA

## Local

Pode ocorrer em acampamento, interrogatório, ruína menor, campo de batalha ou comunicação interceptada.

## Função da Cena para o Mestre

Introduzir a divisão interna dos Hell Raiders. Sem isso, a Cátedra vira só gangue inimiga. Com isso, os PCs têm alavanca.

## Candidatos

- Mara Sable, se a campanha quiser uma porta política.
- Var-Kel Thul-Apex, se quiser uma porta de honra brutal.
- Rourke Venn-Apex, se quiser negociar com liderança perigosa.
- Um Extra não infectado, se for cedo demais para Wild Cards.

## Conflito

O Raider não é inocente. Ele aceita saque, violência e abertura de ruínas. O problema é que ele começa a perceber que a missão atual não pertence mais só ao chapter.

## Desafio Mecânico

Pode ser Social Conflict, Test social, interrogatório, ou cena sem mecânica formal se surgir organicamente.

## Revelações

| Camada | Informação |
|---|---|
| Mínima | nem todos os Raiders falam a mesma língua ritual |
| Provável | alguns acham que o Core será usado de modo errado |
| Profunda | há infectados dentro do chapter |
| Opcional | nomes: Selene, Malachai ou Caine |

# KEY SCENE 6 — DUAS PEÇAS, UMA DIREÇÃO

## Gatilho

Os PCs têm o Mainframe mais uma segunda fonte:

- Sinal Eco;
- Core;
- dados roubados de Hell Raiders;
- ou raises fortes na reconstrução.

## Função da Cena para o Mestre

Fazer a matriz de Outpost-87 funcionar em mesa.

## DT-OUTPOST — Reconstruir o Vetor

### Estrutura Recomendada

- 4 rounds;
- meta de 5 sucessos;
- +1 sucesso automático se os PCs têm Mainframe + Sinal Eco;
- -1 ou complicação se falta a Antena-Telemetria;
- raises revelam precisão, prova técnica ou fragmento de Victor Walker.

### Skills Possíveis

Electronics, Science, Repair, Occult, Notice, Common Knowledge, Survival com Dentinho, Spirit contra ruído temporal.

## Resultados

| Resultado | Consequência |
|---|---|
| Sucesso com raise | Outpost-87 existe; direção e coordenada parcial confiável |
| Sucesso | Outpost-87 existe; direção aproximada a leste/Véu |
| Sucesso parcial | nome confirmado, direção instável |
| Falha | sabem que há destino, mas não conseguem usá-lo ainda |
| Falha crítica | dados atraem atenção ou ativam eco indesejado |

# Fechamento do Ato II

Ao final do Ato II, registre:

| Pergunta | Resposta esperada |
|---|---|
| O Core foi protegido, perdido ou está em disputa? | define entrada do Ato III |
| Os Tarekhs são inimigos, aliados, neutros ou credores? | define Sinal Eco futuro |
| Shizuka apareceu diretamente? | define carga emocional |
| Algum Raider hesitou? | define possibilidade de cisão |
| Outpost-87 virou hipótese? | sim, idealmente |

Estado final ideal:

> Os PCs percebem que a busca pelos fragmentos virou uma guerra de interpretação. Cada facção não quer apenas possuir o satélite; quer dizer o que ele significa.
