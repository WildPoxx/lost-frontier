# Junkyard Blaster Pack — Outsiders: Lost Frontier

Pack de itens JSON para Foundry VTT / SWADE 5.2.6.

## Itens incluídos

1. Junkyard Blaster
2. Junkyard Blaster, Heavy
3. Junkyard Buzzspitter
4. Junkyard Blaster Rifle
5. Junkyard Rattlecarbine
6. Junkyard Breacher
7. Junkyard Heavy Blaster
8. Junkyard Lance
9. Axyon Ghost-Splice Blaster
10. Abutre Scrapstorm

## Nota de design

Todos os itens são derivados dos blasters do Science Fiction Companion, mas rebaixados, corrompidos ou lateralmente modificados para refletir a lógica de Rust Hollow: sucata funcional, Pulvis, emendas instáveis de Teleritha e risco operacional.

A base conceitual usa:
- Mágyon para ignição e aceleração;
- Osmérion para compatibilizar peças incompatíveis;
- Veyra para alinhamento e cadência;
- Pulvis como alimentação degradada;
- Aethera apenas em protótipo raro.
